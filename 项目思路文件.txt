数据库设计

小说表

- id : 主键，自增ID
- title : 小说标题（必填）
- author : 作者名称
- description : 小说简介
- cover_url : 封面图片URL
- status : 小说状态（ongoing进行中、completed已完成、paused暂停）
- total_chapters : 总章节数
- progress_chapters : 进度章节数（已处理的章节数）
- visual_style : 视觉风格（漫画风、写实风、水彩风等）
- art_style : 艺术风格（日式、美式、中式等）
- background_setting : 背景设置（现代都市、古代仙侠、未来科幻等）
- style_prompt : 风格提示词（用于AI生成的详细风格描述）
- genre : 小说类型（玄幻、仙侠、都市、科幻等）
- language : 语言类型（中文、英文等）
- word_count : 总字数
- publish_date : 发布日期
- source_url : 来源链接
- import_status : 导入状态（pending等待导入、imported已导入、failed导入失败）
- analysis_status : 分析状态（pending等待分析、analyzed已分析、failed分析失败）
- storyboard_status : 分镜生成状态（pending等待生成、generating生成中、completed已完成、failed生成失败）
- video_status : 视频生成状态（pending等待生成、processing处理中、completed已完成、failed生成失败）
- created_at : 创建时间
- updated_at : 更新时间
- last_processed_at : 最后处理时间
